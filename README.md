---

## The Tech Stack

To build a modern web experience that matches current industry standards, I chose:

- **Nuxt 4:** For file-based routing, component auto-imports, and optimized server-side rendering/static generation.
- **Tailwind CSS v4:** For a custom, utility-first design system allowing rapid creation of responsive layouts.
- **TypeScript:** To ensure strict type safety, decrease runtime errors, and maintain clean code boundaries.

---

## Key Features Implemented

- **Yin-Yang Monochromatic Theme:** A sleek, minimal, high-contrast dark/light mode layout that switches seamlessly with no flash on initial load.
- **Interactive Navigation & Mobile Support:** A glassmorphism sticky header that transitions to a slide-down hamburger menu on mobile screen widths.
- **Animated Hero Banner:** Slowly panning radial-dot grid, glowing background gradient orbs, and interactive call-to-actions.

## Running the Project Locally

If you are a grader, recruiter, or fellow developer wishing to run this project locally, follow these steps:

### 1. Install Dependencies

```bash
npm install
```

### 2. Run in Development Mode

```bash
npm run dev
```

Open `http://localhost:3000` in your web browser.

### 3. Generate Static Production Build

```bash
npm run generate
```

The static output will be written to the `.output/public` directory, ready to be deployed to serverless environments (GitHub Pages, Netlify, Vercel).
